# Урок wallet_service_lesson

## Задача
Тестовое задание. 

## Требования к заданию
>![ER Diagram](images/diagram.png)
- подключить spring-boot-actuator 
- подключить liquibase
- написать сервис wallet для создания кошелька
- написать класс transaction для пополнения и снятия средств с кошелька
- написать config_service, настроить в нем взаимодействие с файлом [wallet-service-default.yaml](https://github.com/AbdulatipA/wallet_config/blob/main/wallet-service-default.yaml) в репозитории
